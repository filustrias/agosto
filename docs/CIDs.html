CIDs

Aqui estão apenas as listas de CIDs usados nos códigos para identificação de casos. Não foram considerados outras regras aqui como exames laboratoriais, medidas de pressão arterial ou prescrição de medicamentos específicos. Estas informações estão nos capítulos específicos de cada morbidade.

📊 VISÃO GERAL

Este documento descreve os 56 critérios diagnósticos implementados para identificação de morbidades crônicas, utilizando múltiplas fontes de dados:

  • CID-10 (Classificação Internacional de Doenças)
  • Medidas clínicas (Pressão arterial, glicemia, HbA1c, eGFR)
  • Prescrições medicamentosas (Classes terapêuticas)
  • Bases de dados externas (HIV, eGFR)

🎯 Princípio de Persistência

Regra geral: Uma vez detectada uma morbidade crônica, ela persiste em todos os registros subsequentes do paciente.

Exceção: Pré-diabetes não persiste se o paciente progredir para diabetes.

# 1. DOENÇAS CARDIOVASCULARES

1.1. HIPERTENSÃO ARTERIAL SISTÊMICA (HAS)

Descrição:

Pressão arterial sistólica ≥ 140 mmHg e/ou diastólica ≥ 90 mmHg em adultos, ou diagnóstico prévio de hipertensão com uso de anti-hipertensivos.

Critérios de Inclusão:

  • CID-10: I10, I11, I12, I13, I15
  • Uso de anti-hipertensivos: IECA, BRA, diuréticos tiazídicos, beta-bloqueadores, bloqueadores de canal de cálcio

Variável:

  • HAS (DATE) - Data do primeiro diagnóstico

Algoritmo de Identificação:

-- Identificação de HAS
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)I1[0-5]') 
    THEN ea.entrada_data 
END) AS HAS,

-- Validação cruzada com medicamentos
CASE 
    WHEN HAS IS NOT NULL 
         OR uso_IECA = TRUE 
         OR uso_BRA = TRUE 
         OR uso_diuretico_tiazidico = TRUE
         OR uso_beta_bloqueador = TRUE
         OR uso_bloqueador_canal_calcio = TRUE
    THEN TRUE 
    ELSE FALSE 
END AS tem_HAS_validado

CIDs Específicos:

CID Descrição
I10 Hipertensão essencial (primária)
I11 Doença cardíaca hipertensiva
I12 Doença renal hipertensiva
I13 Doença cardíaca e renal hipertensiva
I15 Hipertensão secundária

1.2. CARDIOPATIA ISQUÊMICA (CI)

Descrição:

Doença arterial coronariana, incluindo angina, infarto agudo do miocárdio (IAM) e suas sequelas.

Critérios de Inclusão:

  • CID-10: I20-I25
  • Procedimentos: Angioplastia, revascularização miocárdica
  • Uso de AAS/Clopidogrel como indicador adicional

Variável:

  • CI (DATE) - Data do primeiro diagnóstico

Algoritmo de Identificação:

-- Identificação de Cardiopatia Isquêmica
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)I2[0-5]') 
    THEN ea.entrada_data 
END) AS CI,

-- CIDs mais específicos
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)(I21|I22)') -- IAM
    THEN ea.entrada_data 
END) AS data_IAM,

MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)I20') -- Angina
    THEN ea.entrada_data 
END) AS data_angina

CIDs Específicos:

CID Descrição
I20 Angina pectoris
I21 Infarto agudo do miocárdio
I22 Infarto do miocárdio recorrente
I23 Complicações após IAM
I24 Outras doenças isquêmicas agudas do coração
I25 Doença isquêmica crônica do coração

1.3. INSUFICIÊNCIA CARDÍACA CONGESTIVA (ICC)

Descrição:

Síndrome clínica caracterizada pela incapacidade do coração em bombear sangue adequadamente.

Critérios de Inclusão:

  • CID-10: I50
  • Uso de diuréticos de alça como indicador adicional

Variável:

  • ICC (DATE) - Data do primeiro diagnóstico

Algoritmo de Identificação:

-- Identificação de ICC
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)I50') 
    THEN ea.entrada_data 
END) AS ICC,

-- Validação com medicamentos específicos
CASE 
    WHEN ICC IS NOT NULL 
         OR (uso_diuretico_ansa = TRUE AND (CI IS NOT NULL OR HAS IS NOT NULL))
    THEN TRUE 
    ELSE FALSE 
END AS tem_ICC_provavel

CIDs Específicos:

CID Descrição
I50.0 Insuficiência cardíaca congestiva
I50.1 Insuficiência ventricular esquerda
I50.9 Insuficiência cardíaca não especificada

1.4. ACIDENTE VASCULAR CEREBRAL (STROKE)

Descrição:

Evento cerebrovascular agudo (isquêmico ou hemorrágico) e suas sequelas.

Critérios de Inclusão:

  • CID-10: I60-I69, G45 (AIT)

Variável:

  • stroke (DATE) - Data do primeiro diagnóstico

Algoritmo de Identificação:

-- Identificação de AVC/Stroke
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)(I6[0-9]|G45)') 
    THEN ea.entrada_data 
END) AS stroke,

-- Separar por tipo
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)(I63|I64)') -- Isquêmico
    THEN ea.entrada_data 
END) AS stroke_isquemico,

MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)(I60|I61|I62)') -- Hemorrágico
    THEN ea.entrada_data 
END) AS stroke_hemorragico,

MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)G45') -- AIT
    THEN ea.entrada_data 
END) AS AIT

CIDs Específicos:

CID Descrição
I60 Hemorragia subaracnóidea
I61 Hemorragia intracerebral
I62 Outras hemorragias intracranianas não traumáticas
I63 Infarto cerebral
I64 AVC não especificado (isquêmico ou hemorrágico)
I65-I66 Oclusão/estenose de artérias cerebrais
I67 Outras doenças cerebrovasculares
I68 Transtornos cerebrovasculares em doenças classificadas em outra parte
I69 Sequelas de doenças cerebrovasculares
G45 Acidentes isquêmicos cerebrais transitórios (AIT)

1.5. ARRITMIAS CARDÍACAS

Descrição:

Distúrbios do ritmo cardíaco, incluindo fibrilação atrial, flutter, taquicardias e bradicardias.

Critérios de Inclusão:

  • CID-10: I47-I49

Variável:

  • arritmia (DATE) - Data do primeiro diagnóstico

Algoritmo de Identificação:

-- Identificação de Arritmias
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)I4[7-9]') 
    THEN ea.entrada_data 
END) AS arritmia,

-- Fibrilação Atrial especificamente (mais importante)
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)I48') 
    THEN ea.entrada_data 
END) AS fibrilacao_atrial

CIDs Específicos:

CID Descrição
I47 Taquicardia paroxística
I48 Fibrilação e flutter atrial
I49 Outras arritmias cardíacas

# 2. DOENÇAS METABÓLICAS

2.1. DIABETES MELLITUS (DM)

Descrição:

Distúrbio metabólico caracterizado por hiperglicemia crônica, diagnosticado por glicemia de jejum ≥ 126 mg/dL, HbA1c ≥ 6.5%, ou uso de antidiabéticos.

Critérios de Inclusão:

  • CID-10: E10-E14
  • HbA1c ≥ 6.5%
  • Glicemia de jejum ≥ 126 mg/dL (2+ medidas)
  • Uso de antidiabéticos: Metformina, insulina, sulfonilureias, iDPP4, iSGLT2, agonistas GLP-1

Variáveis:

  • DM (DATE) - Data do primeiro diagnóstico de diabetes
  • pre_DM (DATE) - Data do diagnóstico de pré-diabetes (se não progrediu para DM)

Algoritmo de Identificação:

-- ========================================
-- IDENTIFICAÇÃO DE DIABETES MELLITUS
-- ========================================

-- Diabetes por CID
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)E1[0-4]') 
    THEN ea.entrada_data 
END) AS DM_por_CID,

-- Diabetes por HbA1c
MIN(CASE 
    WHEN lab.nome_exame LIKE '%HEMOGLOBINA GLICOSILADA%'
         AND SAFE_CAST(lab.resultado AS FLOAT64) >= 6.5
    THEN lab.data_resultado 
END) AS DM_por_A1C,

-- Diabetes por glicemia de jejum (2+ medidas)
MIN(CASE 
    WHEN lab.nome_exame LIKE '%GLICOSE%'
         AND SAFE_CAST(lab.resultado AS FLOAT64) >= 126
         AND (
             SELECT COUNT(*) 
             FROM laboratorio l2 
             WHERE l2.cpf = lab.cpf 
               AND l2.nome_exame LIKE '%GLICOSE%'
               AND SAFE_CAST(l2.resultado AS FLOAT64) >= 126
               AND l2.data_resultado <= lab.data_resultado
         ) >= 2
    THEN lab.data_resultado 
END) AS DM_por_glicemia,

-- Diabetes por uso de medicamentos
MIN(CASE 
    WHEN med.nome_medicamento LIKE ANY (
        '%METFORMINA%', '%INSULINA%', '%GLIBENCLAMIDA%', 
        '%GLICLAZIDA%', '%GLIMEPIRIDA%', '%DAPAGLIFLOZINA%',
        '%EMPAGLIFLOZINA%', '%SITAGLIPTINA%', '%VILDAGLIPTINA%',
        '%LIRAGLUTIDA%', '%SEMAGLUTIDA%'
    )
    THEN med.data_prescricao 
END) AS DM_por_medicamento,

-- ========================================
-- CONSOLIDAÇÃO: Data mais antiga
-- ========================================
LEAST(
    DM_por_CID,
    DM_por_A1C,
    DM_por_glicemia,
    DM_por_medicamento
) AS DM,

-- ========================================
-- PRÉ-DIABETES
-- ========================================

-- Pré-diabetes por HbA1c (5.7-6.4%)
MIN(CASE 
    WHEN lab.nome_exame LIKE '%HEMOGLOBINA GLICOSILADA%'
         AND SAFE_CAST(lab.resultado AS FLOAT64) BETWEEN 5.7 AND 6.4
    THEN lab.data_resultado 
END) AS pre_DM_por_A1C,

-- Pré-diabetes por glicemia de jejum (100-125 mg/dL)
MIN(CASE 
    WHEN lab.nome_exame LIKE '%GLICOSE%'
         AND SAFE_CAST(lab.resultado AS FLOAT64) BETWEEN 100 AND 125
    THEN lab.data_resultado 
END) AS pre_DM_por_glicemia,

-- Consolidação pré-DM
CASE 
    WHEN DM IS NULL  -- Só se não tem diabetes
    THEN LEAST(pre_DM_por_A1C, pre_DM_por_glicemia)
    ELSE NULL  -- Se tem DM, pre_DM = NULL
END AS pre_DM,

-- ========================================
-- LÓGICA DE PROGRESSÃO
-- ========================================

-- Tempo entre pré-DM e DM (se progrediu)
CASE 
    WHEN DM IS NOT NULL AND pre_DM_original IS NOT NULL
    THEN DATE_DIFF(DM, pre_DM_original, DAY)
    ELSE NULL
END AS dias_preDM_ate_DM,

-- Flag de progressão
CASE 
    WHEN DM IS NOT NULL AND pre_DM_original IS NOT NULL
    THEN TRUE
    ELSE FALSE
END AS progrediu_preDM_para_DM

CIDs Específicos:

CID Descrição
E10 Diabetes mellitus tipo 1
E11 Diabetes mellitus tipo 2
E12 Diabetes mellitus relacionado à desnutrição
E13 Outros tipos especificados de diabetes mellitus
E14 Diabetes mellitus não especificado

Subcategorias por Complicações:

  • E1x.0 - Com coma
  • E1x.1 - Com cetoacidose
  • E1x.2 - Com complicações renais
  • E1x.3 - Com complicações oftálmicas
  • E1x.4 - Com complicações neurológicas
  • E1x.5 - Com complicações circulatórias periféricas
  • E1x.6 - Com outras complicações especificadas
  • E1x.7 - Com complicações múltiplas
  • E1x.8 - Com complicações não especificadas
  • E1x.9 - Sem complicações

2.2. OBESIDADE

Descrição:

IMC ≥ 30 kg/m² ou diagnóstico de obesidade por CID-10.

Critérios de Inclusão:

  • CID-10: E66
  • IMC ≥ 30 kg/m² (calculado de peso e altura)

Variáveis:

  • obesidade (DATE) - Data do diagnóstico por CID
  • obesidade_consolidada (DATE) - Data do diagnóstico por CID OU primeira medida com IMC ≥ 30

Algoritmo de Identificação:

-- ========================================
-- OBESIDADE POR CID
-- ========================================
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)E66') 
    THEN ea.entrada_data 
END) AS obesidade,

-- ========================================
-- OBESIDADE POR IMC
-- ========================================

-- Cálculo do IMC
peso_mais_recente / POWER(altura_mais_recente, 2) AS IMC_mais_recente,

-- Data da primeira medida com IMC ≥ 30
MIN(CASE 
    WHEN (peso / POWER(altura, 2)) >= 30
    THEN data_medicao
END) AS obesidade_por_IMC,

-- ========================================
-- CONSOLIDAÇÃO
-- ========================================
LEAST(obesidade, obesidade_por_IMC) AS obesidade_consolidada,

-- Classificação do IMC
CASE 
    WHEN IMC_mais_recente < 18.5 THEN 'Baixo peso'
    WHEN IMC_mais_recente BETWEEN 18.5 AND 24.9 THEN 'Normal'
    WHEN IMC_mais_recente BETWEEN 25.0 AND 29.9 THEN 'Sobrepeso'
    WHEN IMC_mais_recente BETWEEN 30.0 AND 34.9 THEN 'Obesidade Grau I'
    WHEN IMC_mais_recente BETWEEN 35.0 AND 39.9 THEN 'Obesidade Grau II'
    WHEN IMC_mais_recente >= 40.0 THEN 'Obesidade Grau III'
END AS classificacao_IMC

CIDs Específicos:

CID Descrição
E66.0 Obesidade devida a excesso de calorias
E66.1 Obesidade induzida por drogas
E66.2 Obesidade extrema com hipoventilação alveolar
E66.8 Outra obesidade
E66.9 Obesidade não especificada

2.3. DISLIPIDEMIA

Descrição:

Alterações no perfil lipídico (colesterol total, LDL, HDL, triglicerídeos) ou uso de hipolipemiantes.

Critérios de Inclusão:

  • CID-10: E78
  • Colesterol total ≥ 200 mg/dL
  • LDL ≥ 130 mg/dL (ou ≥ 100 em alto risco CV)
  • HDL < 40 mg/dL (homens) ou < 50 mg/dL (mulheres)
  • Triglicerídeos ≥ 150 mg/dL
  • Uso de estatinas ou fibratos

Variáveis:

  • dislipidemia_por_CID (DATE) - Data do diagnóstico por CID
  • dislipidemia (DATE) - Data do diagnóstico (CID ou laboratorial)

Algoritmo de Identificação:

-- ========================================
-- DISLIPIDEMIA POR CID
-- ========================================
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)E78') 
    THEN ea.entrada_data 
END) AS dislipidemia_por_CID,

-- ========================================

# 3. DOENÇAS RENAIS

3.1. INSUFICIÊNCIA RENAL CRÔNICA (IRC) / DOENÇA RENAL CRÔNICA (DRC)

Descrição:

Perda progressiva e irreversível da função renal, definida por TFG < 60 mL/min/1.73m² por ≥ 3 meses ou presença de lesão renal.

Critérios de Inclusão:

  • CID-10: N18, N19
  • TFG (CKD-EPI) < 60 mL/min/1.73m² em 2+ medidas com ≥ 90 dias de intervalo
  • Creatinina persistentemente elevada

Variáveis:

  • IRC (DATE) - Data do primeiro diagnóstico
  • estagio_DRC (STRING) - Estágio da DRC: G1, G2, G3a, G3b, G4, G5
  • TFG_CKD_EPI (FLOAT) - Taxa de filtração glomerular estimada

Algoritmo de Identificação:

-- ========================================
-- CÁLCULO DA TFG (CKD-EPI)
-- ========================================

-- Fórmula CKD-EPI (simplificada)
CASE 
    -- Mulheres, creatinina ≤ 0.7
    WHEN genero = 'feminino' AND creatinina_mais_recente <= 0.7
    THEN 144 * POWER(creatinina_mais_recente / 0.7, -0.329) * POWER(0.993, idade)
    
    -- Mulheres, creatinina > 0.7
    WHEN genero = 'feminino' AND creatinina_mais_recente > 0.7
    THEN 144 * POWER(creatinina_mais_recente / 0.7, -1.209) * POWER(0.993, idade)
    
    -- Homens, creatinina ≤ 0.9
    WHEN genero = 'masculino' AND creatinina_mais_recente <= 0.9
    THEN 141 * POWER(creatinina_mais_recente / 0.9, -0.411) * POWER(0.993, idade)
    
    -- Homens, creatinina > 0.9
    WHEN genero = 'masculino' AND creatinina_mais_recente > 0.9
    THEN 141 * POWER(creatinina_mais_recente / 0.9, -1.209) * POWER(0.993, idade)
    
    ELSE NULL
END AS TFG_CKD_EPI,

-- ========================================
-- ESTADIAMENTO DA DRC
-- ========================================

CASE 
    WHEN TFG_CKD_EPI >= 90 THEN 'G1'   -- Normal ou aumentada
    WHEN TFG_CKD_EPI BETWEEN 60 AND 89 THEN 'G2'   -- Levemente diminuída
    WHEN TFG_CKD_EPI BETWEEN 45 AND 59 THEN 'G3a'  -- Leve a moderadamente diminuída
    WHEN TFG_CKD_EPI BETWEEN 30 AND 44 THEN 'G3b'  -- Moderada a gravemente diminuída
    WHEN TFG_CKD_EPI BETWEEN 15 AND 29 THEN 'G4'   -- Gravemente diminuída
    WHEN TFG_CKD_EPI < 15 THEN 'G5'   -- Falência renal
    ELSE NULL
END AS estagio_DRC,

-- ========================================
-- IDENTIFICAÇÃO DE IRC
-- ========================================

-- Por CID
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)N1[89]') 
    THEN ea.entrada_data 
END) AS IRC_por_CID,

-- Por TFG < 60 persistente (2+ medidas com ≥ 90 dias)
MIN(CASE 
    WHEN TFG_CKD_EPI < 60
         AND (
             SELECT COUNT(*) 
             FROM creatininas c2 
             WHERE c2.cpf = c.cpf 
               AND c2.TFG < 60
               AND DATE_DIFF(c2.data, c.data, DAY) >= 90
               AND c2.data <= c.data
         ) >= 2
    THEN data_creatinina
END) AS IRC_por_TFG,

-- Consolidação
LEAST(IRC_por_CID, IRC_por_TFG) AS IRC,

-- ========================================
-- PROGRESSÃO DA DRC
-- ========================================

-- Velocidade de perda de TFG
(TFG_penultimo - TFG_mais_recente) / 
    (DATE_DIFF(data_creatinina_mais_recente, data_creatinina_penultimo, DAY) / 365.25) 
    AS delta_TFG_ano,

-- Classificação de progressão
CASE 
    WHEN delta_TFG_ano < 2 THEN 'estável'
    WHEN delta_TFG_ano BETWEEN 2 AND 5 THEN 'progressao_lenta'
    WHEN delta_TFG_ano > 5 THEN 'progressao_rapida'
END AS progressao_DRC

CIDs Específicos:

CID Descrição

N18.1 Doença renal crônica, estágio 1
N18.2 Doença renal crônica, estágio 2
N18.3 Doença renal crônica, estágio 3
N18.4 Doença renal crônica, estágio 4
N18.5 Doença renal crônica, estágio 5
N18.8 Outra doença renal crônica

# 12. DISTÚRBIOS NUTRICIONAIS

12.1. DESNUTRIÇÃO

Descrição:

Deficiência nutricional proteico-calórica.

Critérios de Inclusão:

  • CID-10: E40-E46
  • IMC < 18.5 kg/m² persistente

Variável:

  • desnutricao (DATE) - Data do primeiro diagnóstico

Algoritmo de Identificação:

-- ========================================
-- DESNUTRIÇÃO POR CID
-- ========================================
MAX(CASE 
    WHEN REGEXP_CONTAINS(ea.condicoes_avaliadas, r'(?i)E4[0-6]') 
    THEN ea.entrada_data 
END) AS desnutricao_por_CID,

-- ========================================
-- DESNUTRIÇÃO POR IMC
-- ========================================
MIN(CASE 
    WHEN IMC_mais_recente < 18.5
         AND (
             SELECT COUNT(*) 
             FROM medidas m2 
             WHERE m2.cpf = m.cpf 
               AND (m2.peso / POWER(m2.altura, 2)) < 18.5
               AND m2.data_medicao <= m.data_medicao
         ) >= 2  -- Persistente
    THEN data_medicao
END) AS desnutricao_por_IMC,

-- Consolidação
LEAST(desnutricao_por_CID, desnutricao_por_IMC) AS desnutricao

E42 Kwashiorkor marasmático
